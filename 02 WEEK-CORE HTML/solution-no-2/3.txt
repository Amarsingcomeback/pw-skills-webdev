Attributes:

In HTML, attributes provide additional information about an element. They are usually included in the opening tag of the element.

Example: <a href="https://pwskills.com/" >Go To pw skills</a>. In this example, the a element is used to create a hyperlink. The href attribute specifies the URL that the hyperlink points to. The attribute values are enclosed in quotes.
